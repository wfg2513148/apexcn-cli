# apexcn-cli v0.80.4

This documentation patch gives first-time users a visual path from the APEX Chinese Community website to a working local CLI profile.

## Changes

- Introduces the APEX Chinese Community and its four primary content sections with a real anonymous homepage screenshot.
- Documents the exact API Key path: sign in, open the account menu, choose **API Key 管理**, and copy the current Key.
- Adds a real API Key management screenshot with the account identity and the entire Key redacted.
- Explains that **重新生成** revokes the previous Key, so it should be used only for intentional rotation.
- Connects the web onboarding flow directly to `apexcn -apikey "YOUR_API_KEY"` and the post-configuration account check.
- Adds an automated documentation check that keeps both screenshot assets present and non-empty.

## Verification

- Verified the public homepage and authenticated API Key management flow in real browser sessions on `https://oracleapex.cn/`.
- Inspected both final screenshot assets at original resolution and confirmed that they contain no account identity or API Key characters.
- Documentation consistency tests, TypeScript build, full automated tests, release checks, and roadmap checks passed.
