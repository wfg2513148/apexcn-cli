# apexcn-cli v0.80.1

This patch makes one-click installation shorter, safer, and independent from authentication.

## Changes

- Replaced the public installer option matrix with zero-argument macOS/Linux and Windows installers.
- Removed API-key handling, account checks, dependency auto-installation, source checkout modes, prompts, and public installer environment switches.
- Always installs the CLI launcher and user-level agent skills.
- Always verifies `apexcn-cli.tgz` against the published SHA-256 checksum; verification cannot be skipped.
- Updates a recognizable older shell-resolved `apexcn-cli` launcher to avoid silently running a stale installation.
- Rejects placeholder, non-ASCII, whitespace-containing, and other invalid API tokens during the separate authentication step.
- Documents the separate install-then-authenticate flow and the minimal one-line commands.
