# apexcn-cli 小白用户手册

这份手册给在 AI 工具里使用 apexcn-cli 的用户。你不用记命令，也不用研究参数；把需求说清楚，让 AI agent 去查、去总结、去整理，重要操作发布前让它先给你确认。

## 1. 第一次使用

先准备好 APEX 中文社区 API key，然后对 AI 说：

> 请帮我安装并配置 apexcn-cli。我会提供 APEX 中文社区 API key。安装完成后，请验证账号、板块列表和搜索能力是否可用。

如果你已经安装过，可以说：

> 请检查 apexcn-cli 是否已经可用，并确认当前登录的是哪个社区账号。不要输出完整 API key。

## 2. 搜索社区帖子

> 请用 apexcn-cli 搜索“APEX REST API”相关帖子，帮我总结前 5 条，并带上每条帖子的真实链接和原文链接。

> 请搜索“ORDS 认证失败”相关内容，把结果按“最可能有用、可以补充阅读、相关性一般”分组。

> 请搜索“APEX JSON_TABLE”的帖子，按适合新手阅读的顺序列出来。

## 3. 查看和总结某个帖子

> 请打开帖子 30549，帮我总结它主要讲了什么，列出关键步骤、注意事项和真实链接。

> 请读取这个帖子，并判断它是否适合解决我现在的问题：我在 APEX 调用 REST API 时拿不到返回 JSON。

## 4. 查账号和板块

> 请查看我当前 apexcn-cli 登录的是哪个社区账号。

> 请列出 APEX 中文社区可以发帖的板块，并告诉我每个板块适合发什么内容。

## 5. 让社区知识回答问题

> 请用 apexcn-cli 根据社区内容回答：Oracle APEX 如何调用 REST API？请列出参考帖子链接。

> 请根据社区已有内容，整理一份 ORDS OAuth2 Bearer Token 排查清单。

## 6. 起草帖子

> 我想在社区提问：APEX 调用 REST API 返回 401。请先帮我搜索相关帖子，再帮我起草一篇清楚的问题描述。先不要发布，等我确认。

> 请根据下面的问题帮我写一篇社区求助帖，包含背景、已尝试方法、报错信息、期望结果和标签。写好后先给我看，不要直接发布。

## 7. 发布帖子

> 请把我确认后的内容发布到合适的板块。发布前先告诉我你准备选择哪个板块、标题是什么、正文是什么、标签是什么，等我确认后再发布。

> 请发布这篇帖子。发布成功后，把帖子真实链接发给我。

## 8. 修改帖子

> 请打开我的帖子 30549，帮我把标题和正文调整得更清楚。先给我预览修改后的版本，不要直接保存。

> 请把刚才确认过的版本更新到帖子 30549，并返回帖子真实链接。

## 9. 删除帖子

> 请查看帖子 30549 的标题和内容，确认它是不是我想删除的那篇。先把标题、作者、板块和链接发给我确认。

> 我确认删除帖子 30549。请按安全流程删除，并把删除结果告诉我。

## 10. 回复帖子

> 请帮我给帖子 30549 起草一条回复，语气友好，补充我的解决方法。先给我预览，不要直接发布。

> 请把刚才确认的回复发布到帖子 30549，并返回帖子真实链接。

> 请回复帖子 30549 中的某条回复，说明我补充的实验结果。先预览再发布。

## 11. 修改和删除回复

> 请查看我的回复 201480，帮我润色一下。先给我修改后的版本，不要直接保存。

> 请把确认后的内容更新到回复 201480。

> 请删除回复 201480。删除前先确认它的内容确实是我想删除的那条。
